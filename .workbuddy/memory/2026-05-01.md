# 2026-05-01

## SEO 优化 - Shell 脚本文档项目

为 VitePress 文档网站完成了全面的 SEO 优化，包括：

### 1. Meta 标签优化
- 添加了基础 SEO meta 标签（author, keywords, robots, copyright 等）
- 配置了 Open Graph 标签用于社交媒体分享
- 配置了 Twitter Card 标签

### 2. 动态 SEO 生成
- 使用 `transformHead` 钩子动态生成每个页面的 SEO 标签
- 为每个页面生成唯一的 canonical URL
- 动态生成 Open Graph 和 Twitter Card 标签

### 3. 结构化数据 (JSON-LD)
- 为首页添加 WebSite 类型结构化数据
- 为文档页面添加 TechArticle 类型结构化数据
- 包含作者、发布者、描述等信息

### 4. Sitemap 生成
- 创建了自定义 `generate-sitemap.cjs` 脚本
- 自动生成包含所有页面的 sitemap.xml
- 支持多语言 hreflang 标签

### 5. Robots.txt
- 在 `docs/public/robots.txt` 创建了爬虫配置文件
- 配置了允许和禁止爬取的目录
- 添加了 Sitemap 位置引用

### 修改的文件
- `docs/.vitepress/config.ts` - 主配置，添加 SEO 标签和 transformHead 钩子
- `docs/.vitepress/generate-sitemap.cjs` - Sitemap 生成脚本
- `docs/public/robots.txt` - 新建，爬虫配置
- `package.json` - 更新 docs:build 脚本

### 注意事项
- Base URL 设置为 `https://shit.sdcom.asia`，可根据需要修改 `config.ts` 中的 `baseUrl` 变量
- sitemap 生成需要先运行构建命令
